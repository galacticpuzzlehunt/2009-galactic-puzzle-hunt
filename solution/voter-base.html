Solution: Voter Base

Step 1. images

The top of this puzzle seems to be blank, but there’s actually a transparent image in it. The URL is voter-base.png. Since this is the title of the puzzle, this is an indication that other images might exist in the same place, and you’ll have to deduce their URLs. In fact, those URLs are the other twelve puzzles in this round.

Step 2. Random Anagrams

Once you have found all of them, you’ll have to unscramble the anagrams. While a little long, they should all be pretty simple to figure out once you intuit some of the key words. Each cluephrase begins with THE WORD YOU ARE LOOKING FOR IS and defines an answer.

Step 3. Series of Tubes

Now we finally get to use the submission box! You likely tried putting some words in already, but probably haven’t found anything that passes. The answers from step 2, however, all do. A successful input produces one letter at the end of the program, but it isn’t reported back to you. Using the clues given on incorrect input, you can deduce what the program is doing, and thus find all the letters. Refer to the comments in the source code for a full explanation.
