Solution: Stationary

We first notice that the pictures appear to have been taken in various subway stations across the world. The main chunk of this puzzle is identifying all the cities and all the stops that have been photographed.

Once the stops are identified, it might be noted that the order of the images is alphabetical order by station name. This clues that each city is a chunk and then we need to reorder.

Because cities are most obviously ordered by location around the world, we reorder by that. Now we need to get a letter from each metro system. To do that, we take everyone’s favorite pangram, “VEX WALTZ, QUICK NYMPH — BIG FJORDS!”, and order the stations using that as an alternate alphabetical order. Next we connect the dots between adjacent stations in each city, spelling letters, eventually getting the cluephrase HANKS TRAIN FILM. Thus, the answer is POLAR EXPRESS.
